Create and draw a ball

  1. The first task is to create a Ball
  2. Copy the following code to the class Ball{}
constructor() {
  this.x = 100;
  this.y = 100;
  this.vx = 5;
  this.vy = 2;
  this.radius = 25;
  this.colour = 'blue';
}
  1. The next task is to draw a ball
  2. Copy the following code to the same class Ball{} after the ball is drawn
draw() {
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  ctx.beginPath();
  ctx.arc(this.x, this.y, this.radius, 0, Math.PI * 2, true);
  ctx.closePath();
  ctx.fillStyle = this.colour;
  ctx.fill();
  }
  1. Notice that the draw() function actually draws a circle at the new coordinates without removing the previously drawn circles. To erase the old circles, clearRect() method is used at the start of the draw() function so that it clears out the previous circle before it draws the new one.

Animation

  1. Examine the canvas and the definition of the ball object
  2. Replace ball.draw(); by the following code:
function drawBall() {
  ball.x += ball.vx;
  ball.y += ball.vy;
  ball.draw()
  window.requestAnimationFrame(drawBall);
}
